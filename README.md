## نظرة عامة

دليل الدورات القرآنية هو منصة تجمع إعلانات الدورات والبرامج القرآنية في المملكة العربية السعودية.

## التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|-----------|
| Next.js 16 (App Router) | إطار العمل الرئيسي |
| TypeScript | لغة البرمجة |
| Tailwind CSS v4 | التنسيق |
| Supabase | قاعدة البيانات، التخزين، المصادقة |
| PostgreSQL | قاعدة البيانات |
| Zod | التحقق من البيانات |
| React Hook Form | إدارة النماذج |
| Cloudflare Turnstile | CAPTCHA |
| Vitest | الاختبارات |
| Vercel | النشر |

## إعداد Supabase

1. اذهب إلى supabase.com وأنشئ مشروعًا جديدًا
2. في SQL Editor، شغّل ملف: supabase/migrations/001_initial_schema.sql
3. شغّل ملف البيانات التجريبية: supabase/migrations/002_seed_data.sql
4. أنشئ حاوية تخزين باسم 'program-images' وفعّل الوصول العام
5. في Authentication > Users، أنشئ حساب المدير

## متغيرات البيئة

انسخ .env.example إلى .env.local وعدّل القيم:

NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
SUPABASE_SERVICE_ROLE_KEY=...
NEXT_PUBLIC_TURNSTILE_SITE_KEY=...
TURNSTILE_SECRET_KEY=...
NEXT_PUBLIC_SITE_URL=...

## التشغيل محليًا

npm install
cp .env.example .env.local
npm run dev

## النشر على Vercel

1. ارفع الكود إلى GitHub
2. اذهب إلى vercel.com وأنشئ مشروعًا جديدًا
3. أضف متغيرات البيئة
4. انشر

## الوظائف المكتملة

- الصفحة الرئيسية مع Hero وبحث وفلاتر
- بطاقات الدورات مع جميع المعلومات
- صفحة تفاصيل كاملة لكل دورة
- نموذج إضافة دورة متكامل مع رفع الصور
- عداد المشاهدات بمنع التكرار
- نظام الإبلاغ عن الإعلانات
- لوحة تحكم محمية بالمصادقة
- إدارة الإعلانات (إخفاء/حذف/استعادة)
- إدارة البلاغات
- Sitemap و robots.txt
- دعم جميع المناطق السعودية الـ 13
- التحقق من البيانات على الخادم والعميل
- Rate Limiting و CAPTCHA
- بيانات تجريبية (9 دورات)
- اختبارات أساسية
